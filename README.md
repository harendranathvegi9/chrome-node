# chrome-node

[![npm](https://img.shields.io/npm/v/chrome-node.svg?style=flat-square)]()

Programmatically control Chrome browser.

## Getting started

### Installation

```sh
npm i chrome-node
```

### Example

```js
const chrome = require('chrome-node');

// Attach to an existing instance
// or launch a new one
```
